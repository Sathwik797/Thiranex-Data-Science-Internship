# Customer Churn Prediction using Machine Learning

## Overview

This project predicts customer churn using machine learning models trained on the IBM Telco Customer Churn dataset.

## Project Workflow

- Data Understanding
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Preprocessing
- Model Building
- Model Evaluation
- Feature Importance
- Model Saving

## Models Used

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

## Best Model

**Logistic Regression**

Accuracy: **80.38%**
