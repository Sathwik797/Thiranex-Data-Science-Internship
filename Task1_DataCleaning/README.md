# Data Cleaning & Visualization using Titanic Dataset

## 📌 Project Overview
This project was completed as part of the Thiranex Data Science Internship.

The objective is to clean the Titanic dataset, handle missing values, detect outliers, and visualize important patterns using Python.

- Jupyter Notebook
