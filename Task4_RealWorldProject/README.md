# Retail Sales Analysis and Sales Prediction

## Overview

This project analyzes a retail sales dataset and builds machine learning models to predict sales.

## Project Workflow

- Data Understanding
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Model Building
- Model Evaluation
- Business Insights

## Models Used

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

## Best Model

**Random Forest Regressor**

**R² Score:** 0.597

- Matplotlib
- Seaborn
- Scikit-learn
- Joblib
- Jupyter Notebook
