# Netflix Movies and TV Shows - Exploratory Data Analysis

## Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset to identify trends, content distribution, and business insights.

## Project Workflow

- Data Understanding
- Data Cleaning
- Exploratory Data Analysis
- Business Insights
- Conclusion

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Visualizations

- Movies vs TV Shows
- Content Added by Year
- Top Countries
- Ratings Distribution
- Release Year Distribution
- Top Directors
- Top Genres
- Monthly Content Added
- Movies vs TV Shows by Year

## Dataset

Netflix Movies and TV Shows Dataset
